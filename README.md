# 📖 [Hello React Rails] <a name="about-project"></a>

Hello Rails React is a Rails and React.js app showcasing Webpack integration. It fetches and displays random greetings from a Postgres database. Perfect for starting a Rails and React project with Webpack bundling.

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need: any web-browser.

### Setup

Clone this [repository](https://github.com/Amen-Musingarimi/hello-rails-react.git) to your desired folder.

### Install

This project does not require installation.

### Usage

**To run the project follow the following instruction:-**

- Clone this [repository](https://github.com/Amen-Musingarimi/hello-rails-react.git) in your local machine with correct branch.
- Run the following commands:-
- **rails db:migrate**
- **rails db:seed**
- **./bin/dev**
